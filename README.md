# StepWise

A lightweight, browser-based hierarchical task management application. StepWise helps you break down projects into nested tasks, track progress, and organize your work visually.

## Features

- **Hierarchical Task Structure**: Create nested tasks with unlimited depth to break down complex projects
- **Status Tracking**: Track tasks as New, In Progress, or Done with automatic parent status updates
- **Progress Visualization**: See completion percentages at every level of your task hierarchy
- **Drag and Drop**: Reorganize tasks by dragging and dropping between parent nodes
- **Keyboard Navigation**: Navigate and manage tasks efficiently with keyboard shortcuts
- **Search**: Quickly find tasks by name or description
- **Undo/Redo**: Full history support for all changes
- **Import/Export**: Save and load your task lists as JSON files
- **No Backend Required**: Runs entirely in the browser with no server dependencies

## Usage

Open [StepWise.html](StepWise.html) in any modern web browser.

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| Arrow Up/Down | Navigate between siblings |
| Arrow Left | Go to parent |
| Arrow Right | Go to first child |
| Space | Expand/collapse node |
| Enter | Add child node |
| F2 | Rename node |
| Delete | Remove node |
| 1/2/3 | Set status (New/In Progress/Done) |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+F | Search |
